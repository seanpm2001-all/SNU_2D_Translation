## ស្ថានភាពកម្មវិធី

រាល់ស្នាដៃរបស់ខ្ញុំគឺមិនមានការរឹតត្បិតខ្លះទេ។ DRM (** D ** igital ** R ** estrictions ** M ** anagement) មិនមាននៅក្នុងស្នាដៃរបស់ខ្ញុំទេ។

! [DRM-free_label.en.svg] (DRM-free_label.en.svg)

ផ្ទាំងនេះត្រូវបានគាំទ្រដោយមូលនិធិសូហ្វវែរឥតគិតថ្លៃ។ ខ្ញុំមិនដែលមានបំណងដាក់បញ្ចូល DRM ក្នុងការងាររបស់ខ្ញុំទេ។

ខ្ញុំកំពុងប្រើអក្សរកាត់ "ការគ្រប់គ្រងការរឹតត្បិតឌីជីថល" ជំនួសឱ្យ "ការគ្រប់គ្រងសិទ្ធិឌីជីថល" ដែលត្រូវបានគេស្គាល់ជាទូទៅថាជាមធ្យោបាយទូទៅក្នុងការដោះស្រាយវាមិនពិតគ្មានសិទ្ធិជាមួយ DRM ទេ។ អក្ខរាវិរុទ្ធ "ការគ្រប់គ្រងការរឹតត្បិតឌីជីថល" គឺមានភាពត្រឹមត្រូវជាងមុនហើយត្រូវបានគាំទ្រដោយ [រីឆាតអិមស្ត្រេម៉ាន (RMS)] (https://en.wikipedia.org/wiki/Richard_Stallman) និង [មូលនិធិសូហ្វវែរឥតគិតថ្លៃ (អេហ្វអេហ្វអេហ្វ)) ( https://en.wikipedia.org/wiki/Free_Software_Foundation)

ផ្នែកនេះត្រូវបានប្រើដើម្បីលើកកម្ពស់ការយល់ដឹងអំពីបញ្ហាជាមួយ DRM និងដើម្បីតវ៉ាផងដែរ។ DRM មានបញ្ហាដោយសារការរចនានិងជាការគំរាមកំហែងដ៏ធំដល់អ្នកប្រើប្រាស់កុំព្យូទ័រនិងសេរីភាពផ្នែកទន់។
