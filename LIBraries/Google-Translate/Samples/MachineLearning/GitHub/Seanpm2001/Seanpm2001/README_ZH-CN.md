2015年，我从一个错误的班级切换到了我应该参加的班级，这彻底改变了我的生活：我的2015年中学计算机编程班。

我一直是语言学家。我喜欢所有不同编程语言及其语法的外观。此外，某些项目依赖于某些语言。当我用一种语言创建IDE时，我更喜欢使用该语言来模拟该语言。
